# Hacker News (TypeScript)

Небольшое приложение на React + TypeScript, которое показывает последние новости Hacker News и страницу конкретной новости с деревом комментариев.

## Возможности

### Главная страница
- Показывает 100 новостей списком (отсортировано по дате: самые свежие сверху)
- Для каждой новости отображается:
  - заголовок
  - рейтинг (points)
  - автор
  - время публикации (в виде "сколько времени назад")
  - количество комментариев (descendants)
- По клику на заголовок / comments происходит переход на страницу новости
- Есть кнопка принудительного обновления новостей
- Есть фильтры ленты: top / new / best / ask / show / jobs (через API Hacker News)

### Страница новости
- Показывает:
  - заголовок
  - автора
  - дату/время
  - количество комментариев
  - ссылку на оригинальную новость (source)
- Комментарии отображаются деревом:
  - корневые комментарии загружаются при входе
  - вложенные загружаются по клику на кнопку "Показать ответы"
  - ответы можно раскрывать/скрывать
- Есть кнопка принудительного обновления комментариев
- Есть кнопка "Назад" на главную

## Технологии
- React
- TypeScript
- Redux Toolkit
- React Router
- Axios
- Hacker News Firebase API

## Требования
- Node.js 18+ (рекомендуется)
- npm / yarn / pnpm (ниже примеры для npm)

## Установка и запуск

Клонировать репозиторий:
```bash
git clone <URL_репозитория>
cd <папка_проекта>


Установить зависимости:
npm install

Создать файл окружения .env (если нужен):
Если проект использует базовый URL API через переменную окружения, создай файл .env в корне проекта:
VITE_HN_BASE_URL=https://hacker-news.firebaseio.com/v0

Запуск в режиме разработки:
npm run dev

Сборка
npm run build

Предпросмотр сборки
npm run preview

Структура проекта (упрощенно)
src/pages/ — страницы (главная, страница новости)
src/components/ — компоненты (шапка, элемент новости, комментарии и т.д.)
src/rtk/ — Redux Toolkit store/slices
src/utils/ — утилиты (formatDate, timeAgo, getDomain и т.п.)
src/App.css — общие стили

Примечания:
Отправка комментариев через Hacker News API без авторизации на сайте невозможна, поэтому поле "Add comment" является UI-заглушкой.
